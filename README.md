# Flight Booking System

A robust and scalable flight booking platform that connects airlines with customers, providing a seamless booking experience with real-time flight search, booking management, and airline administration.

## 🚀 Features

### User Features
- **User Authentication**
    - Secure login with JWT tokens
    - Role-based access control
    - Session management

- **Flight Search & Booking**
    - Search flights by source, destination, and date
    - View available seats and pricing
    - Book and manage reservations
    - View booking history

### Airline Management
- **Airline Registration & Approval**
    - New airline registration with verification
    - Admin approval workflow
    - Document verification

- **Flight Operations**
    - Aircraft registration and management
    - Flight scheduling
    - Seat mapping and configuration
    - Real-time flight status updates

### Admin Features
- **User Management**
    - View all users
    - Manage user status
    - Monitor user activities

- **Airline Management**
    - Approve/Reject airline applications
    - Monitor airline operations
    - Manage flight schedules

## 🛠️ Technical Stack

### Backend
- **Framework**: Spring Boot
- **Security**: JWT Authentication
- **API Documentation**: Swagger/OpenAPI
- **Build Tool**: Maven

### Architecture
- Microservices-based architecture
- RESTful API design
- Centralized authentication and authorization
